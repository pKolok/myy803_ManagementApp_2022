Help Page for "Student Registrations"

This is a help page to guide you through the main functionality of the "Student Registrations" page.

Displayed Content

The page "Student Registrations" lists out all students registered in the chosen course. Each student registration comes with the following information:

  • An ID automatically generated which can not change
  • The name of the student
  • The year of studies of the student
  • The semester of studies of the student
  • The course's ID
  • The year, the student registered for this course
  • The student's project grade
  • The student's exam grade
  • The student's final grade

The following actions can be undertaken in this page:

  • Add a new student in the list of student registrations
  • Edit the information of each student registered
  • Remove a student from the list
  • Calculate the final grades for all students
  • Calculate statistics for the final grades of all students
  • Go back to the page of courses

Each of the above are described in more detail below.

Add a new student

On the top right corner of the screen there is the "Add New" button as shown below.

On clicking the button the "Add New Student" form appears in which the user can add all the attributes that make up a student registration. This form is shown below.

The user must necessarily fill in all fields with the right data types. e.g. in the field "Year of Studies" a number must be provided. Failure to provide the correct data types gives rise to a warning message. The fields along with their data types are described below.

  • [Text] The student's name
  • [Integer] The year of studies of the student
  • [Integer] The semester of studies of the student
  • [Integer] The year, the student registered for this course
  • [Decimal] The student's project grade
  • [Decimal] The student's exam grade

On clicking the "Add Student" button the student is added to the list of students. On clicking the "Cancel" button the "Student Registrations" page is shown.

Edit a student registration

In the last column of the table of courses there is the "Edit" button associated with each student registration. The button is shown below.

On clicking the button the "Edit Student" form appears in which the user can edit all the attributes that make up a student registration. The form is pre-filled with the current student registration attributes. This form is shown below.

The user must necessarily have filled in all fields with the right data types. e.g. in the field "Year of Studies" a number must be provided. Failure to provide the correct data types gives rise to a warning message. The user can change as little as one field or none at all. The fields (in order of appearance) along with their data types are described below.

  • [Text] The student's name
  • [Integer] The year of studies of the student
  • [Integer] The semester of studies of the student
  • [Integer] The year, the student registered for this course
  • [Decimal] The student's project grade
  • [Decimal] The student's exam grade

Delete student registration

In the last column of the table of student registrations there is the "Del" button associated with each student. The button is shown below.

On clicking the button, the respective student registration is removed from the list of courses.

Calculate Final Grades

In the bottom right corner of the page there is the "Calculate Final Grades" button as shown below:

On clicking the button the column "Grade" of the table of student registrations is filled out or updated. The "Grade" column stores the students final grade for the given course. The final grade is calculated based on the project - exam weights as follows: Final grade = exam grade * exam weight + project grade * project weight. The exam weight is provided by the user as a course attribute while the project weight is deducted as 1.0 - exam weight.

Calculate grade statistics

In the bottom right corner of the page there is the "Calculate Grade Statistics" button as shown below:

On clicking the button the "Course Grade Statistics" page is show. It contains a table with the course ID in the first column and the following descriptive statistics for all students registered in the given course:

  • Min
  • Max
  • Mean
  • Standard Deviation
  • Variance
  • 95th Percentile
  • Skewness
  • Kurtosis
  • Median
