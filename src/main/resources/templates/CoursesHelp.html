Help Page for "My Courses"

This is a help page to guide you through the main functionality of the "My Courses" page.

Displayed Content

The page My Courses lists out all courses assigned to the logged-in instructor. Each course comes with the following information

  • An ID automatically generated which can not change
  • The course's name
  • The course's description
  • The course's syllabus
  • The year the course is taught
  • The semester the course is taught
  • A list of students who are registered in each course

The following actions can be undertaken in this page:

  • View the list of students registered in each course
  • Add a new course in the list of courses
  • Edit the information of each course
  • Remove a course from the list
  • Log out

Each of the above are described in more detail below.

Show student registrations

In the second last column of the table of courses there is the "Show" button associated with each course. The button is shown below.

On clicking the button the "Students Registration" page is shown. The contents and actions of this page are described in the respective help page.

Add a new course

On the top right corner of the screen there is the "Add New" button as shown below.

On clicking the button the "Add New Course" form appears in which the user can add all the attributes that make up a course. This form is shown below.

The user must necessarily fill in all fields with the right data types. e.g. in the field "Year" a number must be provided. Failure to provide the correct data types gives rise to a warning message. The fields along with their data types are described below.

  • [Text] The course's name
  • [Text] The course's description
  • [Text] The course's syllabus
  • [Integer] The year the course is taught
  • [Integer] The semester the course is taught
  • [Decimal in [0.0-1.0]] The weight of the course's final exam in the total grade

On clicking the "Add Course" button the course is added to the list of course. On clicking the "Cancel" button the "My Courses" page is shown.

Edit a course

In the last column of the table of courses there is the "Edit" button associated with each course. The button is shown below.

On clicking the button the "Edit Course" form appears in which the user can edit all the attributes that make up a course. The form is pre-filled with the current course attributes. This form is shown below.

The user must necessarily have filled in all fields with the right data types. e.g. in the field "Year" a number must be provided. Failure to provide the correct data types gives rise to a warning message. The user can change as little as one field or none at all. The fields (in order of appearance) along with their data types are described below.

  • [Text] The course's name
  • [Text] The course's description
  • [Text] The course's syllabus
  • [Integer] The year the course is taught
  • [Integer] The semester the course is taught
  • [Decimal 0-1] The weight of the course's final exam in the total grade

Delete course

In the last column of the table of courses there is the "Del" button associated with each course. The button is shown below.

On clicking the button, the respective course is removed from the list of courses.

Log Out

In the bottom left corner of page there is the "Log Out" button which is shown below.

On clicking the button, the current user is logged out of the application and the log-in page is shown.
